## 🏇 Jogo de Corrida de Cavalos (Java)
Um jogo de simulação e apostas de corrida de cavalos desenvolvido em Java, rodando diretamente no terminal com uma interface visual dinâmica feita em texto.

## 🚀 Funcionalidades
- Sistema de Apostas: O usuário pode escolher o seu cavalo favorito entre 8 competidores antes da largada.

- 8 Cavalos na Disputa: Corrida emocionante com 8 competidores únicos correndo simultaneamente.

- Output Visual em Tempo Real: A corrida é renderizada linha a linha no terminal, permitindo acompanhar o avanço de cada cavalo (com seus respectivos ícones) até a linha de chegada.

Obs: Lógica Aleatória -> Cada cavalo avança de forma independente e imprevisível a cada rodada.

## 🛠️ Tecnologias Utilizadas
- Linguagem: Java 23

- Interface: Terminal (Console)

🎮 Como Jogar
1) Clone o repositório:

```bash
git clone https://github.com/gabriellyfn/JogoCorridaDeCavalos_JAVA.git
```

2) Abra o projeto na sua IDE de preferência ou compile direto pelo terminal.

3) Execute a classe principal (Main).

4) Escolha o número do cavalo em que deseja apostar.

Torça enquanto os ícones ganham velocidade no terminal!
